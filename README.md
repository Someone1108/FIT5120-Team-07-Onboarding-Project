# FIT5120-Team-07-Onboarding-Project
Team repository for FIT5120 onboarding project development and collaboration

## Project Overview
UVGuard is a web-based platform designed to help Australians stay safe from harmful UV exposure by providing real-time UV alerts, sun safety education, and clothing recommendations.

## Project Goal
The goal of this project is to design and develop a basic web-based platform called UVGuard that helps users monitor UV exposure and adopt safer sun protection behaviours.
The platform aims to provide real-time UV index information, raise awareness about UV risks, and offer practical clothing recommendations based on current UV levels.
Through iterative development, the project will deliver a functional website prototype that demonstrates key features such as UV alerts, educational content, and personalised protection suggestions.

## Key Features (Iteration 1)
- Real-time UV index alerts
- UV awareness education
- Clothing recommendation based on UV level
